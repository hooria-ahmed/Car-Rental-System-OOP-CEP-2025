# OOP_CEP – Car Rental Management System

This is an Object-Oriented Programming (OOP) based Car Rental Management System developed in Python. The system allows users to view available cars, rent them, and manage rental data using CSV files.

## 📁 Project Structure

```
OOP_CEP/
├── Code.py                  # Main application code
├── REPORT.pdf               # Project report and documentation
├── csv_files/
│   ├── Cars.csv             # Data on available cars
│   ├── RentedCars.csv       # Record of rented cars
│   └── users.csv            # User information
├── pictures/
│   ├── Audi.jpg             # Car images
│   └── ...                  # More car images
```

## 🚗 Features

- View available cars with pictures and interior details.
- Rent and return cars.
- Data stored and retrieved using CSV files.
- Basic GUI/CLI implementation (based on code content).
- Images used to enhance the user experience.

## 🛠️ Technologies Used

- Python 3.x
- CSV for lightweight database handling
- File handling and OOP principles

## 📄 How to Run

1. Make sure Python 3 is installed.
2. Place all files in the correct directory structure.
3. Run the program:

```bash
python Code.py
```

## 📌 Notes

- Make sure the CSV files and images are in the right paths.
- This is an educational project focused on demonstrating OOP in Python.
